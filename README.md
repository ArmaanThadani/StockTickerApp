# Assignment 14 Part 2

This is the Express app for the stock ticker search.

## Local Setup

Install dependencies:

```bash
npm install
```

Set your MongoDB connection string for local testing:

```bash
export MONGODB_URI="mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority"
```

Run the app:

```bash
npm start
```

Open:

```text
http://localhost:3000
```

## Heroku Setup

Create a Heroku app from this `Part_2` folder, then add your MongoDB connection string as a config var:

```bash
heroku config:set MONGODB_URI="mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority"
```

The app uses:

- Home view: `/` using `views/home.html`
- Process view: `/process` generated by Express with the MongoDB results
- Database: `Stock`
- Collection: `PublicCompanies`
